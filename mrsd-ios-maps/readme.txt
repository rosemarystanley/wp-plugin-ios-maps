A simple plugin to convert your address to open in Maps App for iOS devices

== Description ==

Convert your addresses to open in Maps App for iOS devices. 

== Installation ==

1. Upload `mrsd-ios-maps` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add shortcode [ios_maps address="Your Address Here" text="Your Text Here (optional)"] on post, page or text widget where you want the link to show


== Screenshots ==

1. Add shortcode with address and optional text attributes. If no text is given the address will show. 
2. Add the widget to any sidebar area
